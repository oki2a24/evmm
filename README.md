# WBS/EVM Ecosystem Project

このプロジェクトは、Gemini CLI を活用して WBS（Work Breakdown Structure）および EVM（Earned Value Management）の作成、運用、分析、教育を統合的に行うためのエコシステムを構築することを目的としています。

## ビジョン
エクセルという親しみやすいインターフェースと、AI（Gemini）という強力な知能を組み合わせることで、プロジェクトマネジメントの高度な手法を誰でも「正しく、簡単に、戦略的に」活用できる環境を提供します。

## プロジェクトの柱
1. **Infrastructure**: 自動計算と整合性が保証された「最強の雛形」の提供。
2. **Operation**: データの自動更新、エラー検知、自動修復。
3. **Education**: 初学者への理論解説と、実データに基づいたメンタリング。
4. **Intelligence**: 推移分析、リスク予測、次の一手の戦略提案。
